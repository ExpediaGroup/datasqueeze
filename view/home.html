Supported Compaction Types

DataSqueeze can perform compaction in two different ways, depending on your use case.

Normal

User provides source and target location. It stores compacted files on target location retaining directory structure.

In-Place

A User provides source and temp location. It stores compacted files on source location and deletes small files from temp.
